Boilerplate

Headings

Element Description Syntax
<h1> Defines the most important header. <h1>This is a main heading.</h1>
<h2> Defines a second-level header. <h2>This is a second-level heading.</h2>
<h3> Defines a third-level header. <h3>This is a third-level heading.</h3>
<h4> Defines a fourth-level header. <h4>This is a fourth-level heading.</h4>
<h5> Defines a fifth-level header. <h5>This is a fifth-level heading.</h5>
<h6> Defines the least important header. <h6>This is a sixth-level heading.</h6>

Container Elements

Element Description Syntax
<div> A container used to group elements together. <div class="container">Content here</div>
<span> A container used for inline grouping. <h2>This is a second-level heading.</h2>

Document Information

Element Description Syntax
<html> Defines the root of an HTML document. <html lang="en">Content here</html>
<head> Contains metadata, links, and external files. <head><title>Document title.</title></h2>
<meta> Specifies metadata like character set, viewport. <meta charset="UTF-8">
<title> Sets the title of the document (displayed in browser). <title>My Webpage</title>

Semantic Elements

Element Description Syntax
<header> Represents a header section of a page. <header><h1>Welcome</h1></header>
<footer> Defines a footer section of the page. <footer><p>Footer content here</p></footer>
<article> Defines a self-contained article. <article><h2>Article Title</h2></article>
<section> Represents a thematic section of content. <section><h2>About Us</h2></section>
<nav> Defines a navigation section for links. <nav><ul><li><a href="#">Home</a></li></ul></nav>

Text Formatting & Inline Text Semantics

Element Description Syntax
<p> Defines a paragraph of text. <p>This is a paragraph.</p>
<b> Defines bold text. <b>This is bold text.</b>
<i> Defines italicized text. <i>This is italic text.</i>
<strong> Defines strong emphasis (bolded text). <strong>Important text</strong>
<em> Defines emphasized text (italicized text). <em>Important emphasis</em>
<mark> Highlights text. <mark>Highlighted text</mark>
<u> Underlines text. <u>Underlined text</u>
<big> Big text. <big>Big text</big>
<small> Small text. <small>Small text</small>
<sup> Superscript text. <sup>Superscript text</sup>
<sub> Subscript text. <sub>Subscript text</sub>
<del> Deleted text. <del>Deleted text</del>

Lists

Element Description Syntax
<ul> Defines an unordered (bulleted) list. <ul><li>Item 1</li><li>Item 2</li></ul>
<ol> Defines an ordered (numbered) list. <ol><li>Item 1</li><li>Item 2</li></ol>
<li> Defines a list item. <li>List item</li>

Tables

Element Description Syntax
<table> Defines a table. <table><tr><td>Data</td></tr></table>
<tr> Defines a table row. <tr><td>Row 1</td></tr>
<th> Defines a table header cell. <th>Header</th>
<td> Defines a table data cell. <td>Data cell</td>

Form

Element Description Syntax
<form> Defines a form for user input. <form action="/submit"><input type="text" name="name"></form>
<input> Defines an input field for user data. <input type="text" name="username">
<button> Defines a clickable button. <button>Click Me</button>
<select> Defines a dropdown menu for selecting an option. <select><option value="1">Option 1</option></select>
<textarea> Defines a multi-line text input field. <textarea rows="4" cols="50"></textarea>

Multimedia

Element Description Syntax
<img> Embeds an image in the document. <img src="image.jpg" alt="Description>
<audio> Embeds audio content. <audio controls><source src="audio.mp3" type="audio/mp3"></audio>
<video> Embeds video content. <video width="320" height="240" controls><source src="movie.mp4" type="video/mp4"></video>

Attributes

Element Description Syntax
class Assigns a class to an element for styling. <div class="container">Content</div>
id Assigns a unique identifier to an element. <div id="header">Header</div>
href Specifies the URL for a hyperlink. <a href="https://example.com>Click Here</a>
src Specifies the source URL for media elements. <img src="image.jpg" alt="image">
alt Provides alternative text for images. <img src="image.jpg" alt="image">
style Defines inline CSS styles for an element. <p style="color:red;">Text</p>

Fonts and Typography

Property Description Example
font-family Specifies the font for the text. font-family: Arial, sans-serif;
font-size Sets the font size. font-size: 16px;
font-weight Defines the thickness of the text. font-weight: bold;
font-style Defines the style of the font (normal, italic, oblique). font-style: italic;
line-height Specifies the space between lines of text. line-height: 1.5;
letter-spacing Controls the space between letters. letter-spacing: 2px;
text-transform Controls capitalization of text (uppercase, lowercase, capitalize). text-transform: uppercase;
text-decoration Adds decoration to text (underline, overline, etc.). text-decoration: underline;

Colors and Backgrounds

Property Description Example
color Sets the text color. color: red;
background-color Sets the background color. background-color: lightblue;
background-image Sets a background image. background-image: url('image.jpg');
background-size Defines the size of the background image. background-size: cover;
background-position Defines the position of the background image. background-position: center;
opacity Sets the opacity level of an element. opacity: 0.5;

Box Model (Spacing, Borders, Sizing)

Property Description Example
padding Adds space inside the element, between the content and its border. padding: 10px;
margin Adds space outside the element, around its border. margin: 20px;
border Defines the border of an element. border: 2px solid black;
border-radius Rounds the corners of an element. border-radius: 10px;
width Sets the width of an element. width: 50%;
height Sets the height of an element. height: 200px;
box-sizing Controls how the width and height of an element are calculated. box-sizing: border-box;

Layout (Positioning and Display)

Property Description Example
display Defines how an element should be displayed (block, inline, flex, etc.). display: flex;
position Specifies the type of positioning (static, relative, absolute, fixed). position: relative;
top, right, bottom, left Specifies the position of an element (used with position). top: 10px; left: 20px;
flex Defines the flexible container and its children in a flexbox layout. display: flex;
grid Defines the grid layout container. display: grid;
justify-content Aligns items along the main axis (in flex or grid layouts). justify-content: space-between;
align-items Aligns items along the cross axis (in flex or grid layouts). align-items: center;
flex-direction Defines the direction of flex items (row, column, etc.). flex-direction: column;

Alignment and Text Styling

Property Description Example
text-align Aligns the text (left, center, right). text-align: center;
vertical-align Aligns inline or inline-block elements vertically. vertical-align: middle;
white-space Controls how whitespace inside an element is handled. white-space: nowrap;

Transitions and Animations

Property Description Example
transition Defines the transition effect when an element's property changes. transition: all 0.3s ease;
animation Defines an animation with keyframes. animation: slide 3s infinite;

Effects (Shadows, Gradients)

Property Description Example
box-shadow Adds shadow to an element. box-shadow: 2px 2px 5px grey;
text-shadow Adds shadow to text. text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
linear-gradient Creates a linear gradient background. background: linear-gradient(to right, red, yellow);
radial-gradient Creates a radial gradient background. background: radial-gradient(circle, red, yellow);

Miscellaneous Properties

Property Description Example
cursor Defines the type of cursor to be displayed (pointer, text, etc.). cursor: pointer;
visibility Specifies if an element is visible or not. visibility: hidden;
z-index Specifies the stack order of elements. z-index: 10;
overflow Defines what happens if content overflows an element's box. overflow: hidden;
